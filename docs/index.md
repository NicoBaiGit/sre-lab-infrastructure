# Bienvenue sur le Lab SRE

Ce site documente l'infrastructure et les procédures de mise en place de mon laboratoire personnel dédié à la pratique du **Site Reliability Engineering (SRE)**.

## Objectifs du Lab

*   **Infrastructure as Code (IaC)** : Automatiser le déploiement des serveurs et des applications.
*   **Kubernetes** : Maîtriser l'orchestration de conteneurs avec K3s.
*   **GitOps** : Utiliser Git comme source unique de vérité pour l'état du système (via ArgoCD).
*   **Observabilité** : Mettre en place le monitoring, le logging et le tracing.

## Structure de la documentation

*   [Guide Lab SRE](setup-lab.md) : La feuille de route étape par étape pour construire le lab.
*   [Configuration Serveur](ubuntu-server.md) : Les détails techniques de l'installation du serveur physique (ThinkPad T420).

!!! info "Matériel"
    Le lab repose principalement sur un **Lenovo ThinkPad T420** recyclé en serveur "Headless" tournant sous Ubuntu Server.
